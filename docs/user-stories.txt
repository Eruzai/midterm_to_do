USER STORIES!

- As a user I want to keep track of the things that I want to do
- As a user I want to be able to categorize my to do list automatically
- As a user I want to be able add items from my list
- As a user I want to be able to remove items from my list
- As a user I want to be able to edit things from my list
- As a user I want to be able to mark an item as completed
- As a user I want to be able to see the items that I completed
- As a user I want to be able to prioritize/reOrder things from my list
- As a user I don't want other people to see/edit/remove things from my list
- As a user I want to see how many items are in a list
- As a user I want to see my percentage of completion for each list
- As a user I want to be able to create my own category (stretch)
